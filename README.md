# HuariqueHub Mobile

Aplicación móvil desarrollada en Kotlin con Android Studio y Jetpack Compose.

## Requisitos

- Android Studio
- JDK 17
- Emulador Android o dispositivo físico
- Conexión a internet

## Cómo ejecutar el proyecto

1. Abrir Android Studio.
2. Seleccionar la carpeta principal del proyecto: `HuariqueHub-App`.
3. Esperar a que termine la sincronización de Gradle.
4. Seleccionar un emulador Android.
5. Ejecutar la app con el botón Run.

## Tecnologías usadas

- Kotlin
- Jetpack Compose
- Retrofit
- Moshi
- OkHttp
- Coil
- osmdroid

## Funcionalidades principales

- Inicio de sesión
- Registro de usuario propietario
- Listado de huariques
- Búsqueda y filtros
- Favoritos
- Perfil de usuario
- Gestión de huariques y promociones

## Notas de desarrollo

La aplicación móvil consume los servicios del backend mediante Retrofit.

Para probar cambios visuales o validaciones simples, se recomienda ejecutar la app en un emulador Android desde Android Studio.

Antes de subir cambios, verificar que el proyecto compile correctamente y que la pantalla de inicio cargue sin errores.
